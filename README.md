# FFX_WorkerDirector

TAB1: Allows you to add new workers (NPC's, Managers, Weapons, Edge Functions) and save and load different profiles).

TAB2:Allows you to use a freecam in game by pressing 0 or 1 and then records the script code with left clicking.

TAB3: Allows you to edit game field dialogue.
